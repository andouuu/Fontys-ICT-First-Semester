 About the game


 This is the main screen of my game.
 It shows every character and it has a button for starting the game,
 the instructions and background change page.

 Instructions


 -You start the game with full happiness meter
 -There is a word that you need to guess by typing letters and revealing them
 -Every time you get a letter wrong your happiness meter goes down
 -When your happiness meter drops to 0 you lose the game and the animal starts crying
 -Have fun!
